# Как выложить сайт Чемпион 24/7

## Вариант 1: Netlify Drop (самый простой)

1. Открой **https://app.netlify.com/drop**
2. Зарегистрируйся (бесплатно) или войди
3. Перетащи папку **«сайт чемпиона»** в окно браузера
4. Готово! Netlify даст тебе ссылку вида `https://random-name.netlify.app`

---

## Вариант 2: Vercel

1. Установи Node.js: https://nodejs.org
2. Открой терминал в папке проекта и выполни:
   ```bash
   npx vercel
   ```
3. Следуй подсказкам (войди в аккаунт Vercel)
4. Получишь ссылку вида `https://твой-проект.vercel.app`

---

## Вариант 3: GitHub + Vercel (автодеплой)

1. Создай репозиторий на GitHub
2. Залей туда файлы проекта
3. Зайди на https://vercel.com → New Project → Import из GitHub
4. При каждом обновлении репозитория сайт будет обновляться автоматически
